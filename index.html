📊 Review Intelligence System

Collect reviews, analyze sentiment, and gain insights in real-time

👋 Welcome

This is a complete review collection and analytics system powered by Zapier, Google Sheets, and Google Apps Script. Collect customer feedback, automatically route positive and negative reviews to separate sheets, and visualize insights through a beautiful dashboard.

🎯 Quick Start

📝 Submit a Review 📊 View Dashboard

✨ Features

📝 Review Form

Beautiful, responsive form for collecting customer feedback with star ratings

🔄 Auto Routing

Zapier automatically routes positive and negative reviews to separate sheets

📊 Dashboard

Real-time analytics with charts, keywords, and smart insights

🚨 Alerts

Get notified of critical sentiment changes and trends

📈 Trends

Track review volume and sentiment over time

🔑 Keywords

Automatically extract and analyze frequently mentioned topics

📚 Documentation

Setup guides and API documentation:

  • Google Apps Script Setup - Create the backend API
  • Dashboard Setup - Deploy to GitHub Pages
  • Google Sheets Setup - Create data structure
  • GitHub Pages Deployment - Host on GitHub

🏗️ Architecture

Review Form → Zapier → Google Sheets (2 tabs) → Google Apps Script API → Dashboard

All data flows through Google services. No backend server required. No CORS issues. Fully serverless architecture.
